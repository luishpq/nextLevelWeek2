Dicas

    - Quebra de linha: alt + Z


1) CSS

    1.1) Classe e Id
        Para atribuir uma classe basta digitar .nomedaclasse
        Para atribuir um ID basta digitas #nomedoID
        Lembre-se que esse nome será utilizado dentro do atributo class.
        O ID deve ser único, já a classe pode ser geral.

    1.2) Box-Sizing
        - Box-sizing: border-box > isso significa que ele vai travar as divs nos tamanhos determinados.

    1.3)Display 
        - Display block: ocupa toda uma linha (div, h1, p)
        - Diplay flex: é flexível. (a, img)

    1.4) Fonte (element1 element2 element3)
        - Element1 é o peso da fonte (grossura) (font-weight)
        - Element2 é o tamanho da Fonte (font-size)
        - Element3 é o nome da fonte (Arial, Times..) (font-family)
